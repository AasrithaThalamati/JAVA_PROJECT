# 📊 Smart Hybrid Number Series (Java Project)

## 📌 Overview

This project is all about creating something a little different from the usual number patterns. Instead of sticking to just one concept, this program mixes **Fibonacci numbers, prime numbers, and simple conditions** to generate a unique and interesting series.

The idea behind this project is to show how multiple concepts can be combined to build something creative while still keeping the logic clear and understandable.

---

## 🚀 What This Program Does

* Generates a custom number series that isn’t commonly found in textbooks
* Uses both **Fibonacci sequence** and **prime numbers** together
* Changes behavior based on whether the step is **even or odd**
* Displays the output neatly in a table
* Gives a quick summary and analysis of the generated numbers

---

## 🧠 How the Logic Works (Simple Explanation)

* We start with two numbers: `1` and `1`

* For every new term:

  * We take the next Fibonacci number
  * We take the next prime number
  * Then we combine them with the previous values

* If the step is:

  * **Even →** we add everything
  * **Odd →** we slightly modify the formula to keep the pattern interesting

This keeps the series dynamic instead of predictable.

When you run the program, it will show:

  * Digit sum

It also gives a summary like:

* Total numbers generated
* Average value
* Highest and lowest numbers
* How the series grows over time
