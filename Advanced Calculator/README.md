# 🧮 Advanced Scientific Calculator (Java)

## 📌 Project Overview

The **Advanced Scientific Calculator** is a console-based Java application designed to perform both basic arithmetic and advanced scientific calculations.
It provides a menu-driven interface, allowing users to perform operations such as trigonometric functions, logarithms, factorials, and more.

---

## 🚀 Features

### 🔢 Basic Operations

* Addition
* Subtraction
* Multiplication
* Division
* Modulus

### 📐 Scientific Operations

* Power (a^b)
* Square Root
* Sine (sin)
* Cosine (cos)
* Tangent (tan)
* Logarithm (base 10)
* Natural Log (ln)
* Factorial

### ⚙️ Additional Features

* Menu-driven interface
* Input validation
* Exception handling (e.g., divide by zero, invalid inputs)
* Continuous execution until exit
