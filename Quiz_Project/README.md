# 🧠 Advanced Quiz Application (Java)

## 📌 Project Overview

The **Advanced Quiz Application** is a console-based Java program designed to conduct multiple-choice quizzes with enhanced features such as time tracking, score calculation, difficulty levels, and replay functionality.
This project demonstrates strong use of Object-Oriented Programming (OOP) principles along with user interaction and input validation.

---

## 🎯 Features

* Multiple-choice quiz system (MCQ)
* Difficulty levels (Easy, Medium)
* Randomized question order
* Timer-based evaluation (10 seconds per question)
* Score and percentage calculation
* Performance feedback (Excellent, Good, Needs Improvement)
* Input validation to prevent errors
* Replay option for multiple attempts

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Classes, Objects, Encapsulation)
* ArrayList (Java Collections Framework)
* Scanner (User Input)
* System Time (Timer Logic)
