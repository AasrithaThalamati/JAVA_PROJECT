# 📚 Library Management System (Java)

## 📌 Project Overview

The **Library Management System** is a console-based Java application designed to manage books efficiently within a library.
It allows users to add, view, search, issue, and return books using a simple menu-driven interface.

This project demonstrates the use of **Object-Oriented Programming (OOP)** concepts along with basic data structures and user interaction.

---

## 🚀 Features

* 📖 Add new books
* 📄 View all books
* 🔍 Search books by title
* 📕 Issue books
* 📗 Return books
* 📊 Track book availability
* 🛡️ Input validation and error handling

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Classes, Objects, Encapsulation)
* ArrayList (Java Collections Framework)
* Scanner (User Input)
