# 🤖 Smart Expense Splitter (Java)

## 📌 Project Overview

The **Smart Expense Splitter** is an advanced console-based Java application designed to manage and optimize shared expenses among multiple users.
Inspired by modern apps like Splitwise, this system not only tracks expenses but also provides **intelligent insights and optimized debt settlements**.

---

## 🚀 Key Features

* 👥 Add and manage multiple users
* 💰 Record shared expenses
* ⚖️ Automatic equal expense splitting
* 📊 Real-time balance tracking
* 💸 **Optimized debt settlement (minimum transactions)**
* 🧾 Expense history tracking
* 🤖 **Smart Insights (AI-style analytics)**
* 🛡️ Input validation and error handling

---

## 🧠 Unique Highlights

* Uses **Greedy Algorithm with Priority Queues** for minimizing transactions
* Provides **AI-like suggestions and spending insights**
* Tracks **top spender and pending debts**
* Simulates real-world financial applications

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Classes, Objects, Encapsulation)
* Collections Framework (HashMap, ArrayList, PriorityQueue)
* Scanner (User Input Handling)
