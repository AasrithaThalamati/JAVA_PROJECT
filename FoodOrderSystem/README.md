# 🍔 Online Food Ordering System (Java)

## 📌 Project Overview

The **Online Food Ordering System** is a simple console-based Java application that allows users to select food items, add them to an order, and view the total bill. This project demonstrates basic Object-Oriented Programming (OOP) concepts and user interaction using a menu-driven approach.

---

## 🎯 Features

* Display food menu
* Add items to order
* View ordered items
* Calculate total bill
* Simple and interactive console interface

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Classes, Objects)
* ArrayList (Collection Framework)
* Scanner (User Input)

---

## 📂 Project Structure

```
FoodOrderingSystem.java
```

## ▶️ How to Run

### Step 1: Compile the program

```
javac FoodOrderingSystem.java
```

### Step 2: Run the program

```
java FoodOrderingSystem
```
